Extended Defunc Interpreter


Defunc is an esoteric functional programming language created by Obvious#5283.
Functions are called by putting the function name on the left and then the arguments to the right.
Functions are defined by naming the new function, choosing variable names, and then writing the return value of the function (which must be another function)
Valid variable names are any function that has not been defined earlier in the file. You can only reference functions defined earlier in the file; recursion allowed.
All tokens are one character. This page uses UTF-8.
Spaces in the code are ignored. Newlines separate functions.
Numbers are standard javascript doubles; don't expect things to work perfectly above 2^53.
Functions with 0 variables are automatically run as soon as they get parsed (value is stored in an object for efficiency)
There are 4 builtin functions:
0 (returns numberical value 0)
+a (returns a+1)
?abtf (if a>b, return t; else return f)
.a (returns and outputs a)
This implementation adds two more builtins for performance purposes (oh also these are code examples):
Arci?ciA+rc+ir (if c>i, return r+c-i; else return r)
*abri?bi*Aar0br+ir (if b>i, return a*(b-i)+r; else return r)
Some extra code examples (not builtin):
Ia?000a (identity)
|ab?abab (max)
